Wheel Offset

This project is centered on developing precise offset calculations for swapping between summer and winter tires in automotive applications, utilizing point cloud data and filtering algorithms to achieve millimeter-level precision.

Wheel Pick & Place

The Wheel Pick & Place project built on the foundation of the previous wheel offset initiative, demonstrating a cost-effective solution to RoboTire's tire changing sequence. By integrating a $300 OakD-SR camera in place of the expensive $12,000 Zivid cameras, the project maintained sub-millimeter precision in lifting wheels from the floor and accurately placing them on car hubs. This venture combined the YOLO model, point cloud data, OpenCV, Open3D, mathematical geometry, and numpy to achieve high precision at a fraction of the cost.

Lug Segmentation

The Lug Segmentation project refined object detection by transitioning from bounding box approaches to precise YOLO-based shape extraction of lug nuts. This advancement enables more accurate and application-specific functionalities at RoboTire.

A* Algorithm

Implemented two variants of the A* algorithm for a PR2 robot to navigate collision-free paths in a given environment, distinguishing between "4-connected" and "8-connected" neighbors to ascertain the shortest route from start to goal.

RRT Algorithm

Implemented the RRT-Connect algorithm for a 6-DOF PR2 robot arm, planning collision-free paths in configuration space with a step size of 0.05rad and a 10% goal bias. Additionally, a shortcut smoothing algorithm was applied over 150 iterations to optimize the path, visually distinguishing the original end-effector path in red and the shortcut-smoothed path in blue in the viewer.

Grasping Manipulation

Implemented friction cone calculation and force closure evaluation for a Robotiq gripper and Kuka iiwa arm in PyBullet, focusing on discretizing the friction cone with varying vector quantities.

Kimera VIO

The project involved creating a real-time Visual Inertial Odometry (VIO) system for a drone, utilizing Kimera VIO and Intel RealSense D455 on a Jetson Xavier platform. It was benchmarked against Vicon's ground truth data to effectively identify dynamic obstacles for efficient path planning and navigation. The system's components were containerized using Docker to streamline the deployment of ROS, RealSense, and Kimera modules on the Jetson Xavier.

VINS Fusion

Implemented a real-time VIO system on a quadrotor using VINS Fusion with Intel RealSense D455 on Jetson Xavier, benchmarked against Vicon for ground truth accuracy. The system integrated Octomap to precisely identify dynamic obstacles, enhancing navigation, and was containerized using Docker for streamlined deployment.

SLAM

Executed 2D and 3D pose estimation using GTSAM for the Intel Research Lab and parking garage datasets, focusing on batch and incremental solutions. This involved implementing pose graph optimization in Python and scripting to visualize both unoptimized and optimized trajectories.

Masked LIOSAM

Engineered a SLAM system by integrating SECOND object detection with LIO-SAM, improving localization accuracy by effectively filtering moving objects in point cloud data, enhancing the capabilities of standard LIO-SAM.

3D bounding Box Estimation

Evaluated 3D bounding box estimation on the KITTI 3D dataset using MobileNetv2 and VGG architectures. Focused on feature extraction from monocular images, with analysis involving training loss and 2D bird's eye view representations, ultimately demonstrating MobileNetv2's enhanced accuracy over VGG.

GANs

Successfully replicated "Region-Based Semantic Factorization in GANs," refining code for facial feature manipulation in images, and conducting evaluations using FID, Masked MSE, and ID metrics.

Phone Detection

Developed a visual object detection system prototype to locate phones using single RGB images, employing YOLO model for detection. Innovatively created YOLO-compatible labels through classical computer vision techniques and OpenCV, instead of standard labeling tools, leading to a unique blend of traditional and contemporary object detection methodologies.
